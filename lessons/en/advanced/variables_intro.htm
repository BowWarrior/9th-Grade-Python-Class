Variable Comparisons

Variables are basically containers that help you and your program remember things.  You use variables when you want to keep track of something.  If you wanted to keep track of how many beepers you picked up, you would use a variable to do this.

Declaring and assigning the Variable a value

You can assign a value to a variable with the assignment statement. The assignment statement looks like this,

Variablename = expression

Where you have the variable name on the left of the = sign and an expression on the right of the = sign. An expression can be a complex combination of other variables and function calls. For now however we will simplify it to be either a value or a combination of a value and a variable. So,

beepCnt = 5

Would assign the value 5 to beepCnt.

It is important to understand that the = sign does not mean what it does in algebra. In algebra, it is a statement of equality. a=b means that a equals b forever and always. When programming, it is an assignment statement and after the statement is executed a will equal b. If something is done later on down the road to b, a will not change.  Think of it as a is assigned the current value of b.

An initial value is usually assigned to a variable at the beginning of the program.  You might see the following line at the beginning of a program or function

beepCnt = 0

Using the Variable (Counting with a Variable in this case)

As you have witnessed with loops, you can execute the same line of code multiple times. For example

 

            while front_is_clear():

                move()

The statement move() is executed multiple times. It does the same thing each time, but the position of Reeborg is different (or varies) each time the statement is executed, that same line of code is able to move Reeborg across the screen. The key here is that where Reeborg moves to depends on where Reeborg currently is.

A very similar situation exists using variables. The code below will count the number of times Reeborg moves.

 

            moveCnt = 0               (This line would go at the beginning of your program)

            while front_is_clear():

                move()

                moveCnt = moveCnt + 1

The first time Reeborg moves, the value of moveCnt is 0. So,

moveCnt = 0 + 1    or            1

The second time Reeborg moves, the value of moveCnt is 1. So,

moveCnt = 1 + 1    or            2

The third time Reeborg moves, the value of moveCnt is 2. So,

moveCnt = 2 + 1    or            3

and so on...
